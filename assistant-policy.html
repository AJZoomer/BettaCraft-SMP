BettaCraft Assistant — Policy & Information

Everything you need to know about how the BettaCraft Assistant works

What It Can Do What It Cannot Do How It Works Privacy Allowed Usage Disallowed Usage Limitations Contact

The BettaCraft Assistant is a friendly, interactive helper built specifically for the BettaCraft SMP website. It helps visitors explore the site, learn about the SMP, and access information quickly.

BettaCraft SMP is a private, creator‑only Minecraft server. The assistant cannot grant access, accept applications, or provide whitelisting.

The assistant is not a real AI model. It is a handcrafted, rule‑based conversational system built entirely in JavaScript. All responses come from predefined logic stored inside the website.

What the Assistant Can Do

  • Provide information about BettaCraft SMP (seasons, members, builds, rules, downloads, history)
  • Help users navigate the website with links and suggestions
  • Engage in light conversation (greetings, small talk, jokes, fun facts)
  • Offer deeper explanations when asked (“tell me more”, “go deeper”)
  • Respond to identity questions (“who are you”, “what are you”, “are you an AI”)
  • Stay safe, friendly, and helpful

What the Assistant Cannot Do

Because BettaCraft SMP is a private creator‑only server, the assistant is not allowed to:

  • Provide access to the server (no whitelisting, invites, or applications)
  • Add players, viewers, or fans to the SMP
  • Share private or internal server information (coordinates, private builds, member‑only content)
  • Perform moderation or administrative actions
  • Provide real‑time server data (TPS, RAM, player count, console logs)
  • Access personal information or track users
  • Give legal, medical, or financial advice
  • Respond to harmful or inappropriate messages

How the Assistant Works

  • Runs entirely in your browser (client‑side)
  • Uses a keyword‑based understanding system
  • Has a short‑term memory of the last 3 messages
  • Includes a tone engine for natural responses
  • Supports deep‑dive explanations
  • Uses time‑aware greetings and idle personality
  • Does not use machine learning or external APIs

Refreshing the page clears the assistant’s memory. No data is stored or transmitted.

Privacy & Data

  • Does NOT collect personal data
  • Does NOT send messages to a server
  • Does NOT store chat logs
  • Does NOT track users
  • Does NOT use cookies

All processing happens locally inside your browser.

Allowed Usage

  • Asking questions about BettaCraft
  • Exploring the SMP through the assistant
  • Using it for fun conversation
  • Navigating the website
  • Learning about seasons, builds, and members

Disallowed Usage

  • Attempting to join the private SMP
  • Requesting whitelisting or access
  • Attempting to exploit or break the assistant
  • Requesting harmful or inappropriate content
  • Using it to impersonate BettaCraft members
  • Trying to bypass server rules
  • Attempting to obtain private or restricted information
  • Using it for harassment or inappropriate messages

Limitations

  • May misunderstand complex questions
  • May ask for clarification
  • Provides simplified explanations
  • Only answers topics it has been programmed with

The assistant is a simulation — not a real AI model.
