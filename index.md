# Welcome to the BettaCraft SMP Official Website
# About:
BettaCraft SMP is a private, whitelisted minecraft server for content creators!
We are a long running SMP, meaning we spend more than a year in the same world before moving on to a new world.
Each season brings fresh ideas, base themes, and new stories to be told!
